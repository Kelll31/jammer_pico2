# KELL31 Jammer - MicroPython версия

Полный порт проекта KELL31 Jammer с C на MicroPython. Проект включает управление дисплеем ILI9341 с тачскрином XPT2046, генерацию сигнала джаммера с различными режимами работы и сохранение настроек.

## Структура проекта

```
micropython/
├── config.py              # Конфигурация оборудования и параметров
├── ili9341.py            # Драйвер дисплея ILI9341 с двойной буферизацией
├── xpt2046.py            # Драйвер тачскрина XPT2046
├── jammer_signal.py      # Генерация сигнала джаммера (4 режима работы)
├── settings.py           # Сохранение и загрузка настроек в JSON
├── ui_manager.py         # UI менеджер с тач-интерфейсом
├── button.py             # Обработка аппаратных кнопок (опционально)
├── main.py               # Главный файл приложения
├── test_project.py       # Тестовый файл для проверки работы
└── README.md             # Эта документация
```

## Особенности реализации

### 1. Драйвер дисплея ILI9341
- Двойная буферизация для плавной анимации
- Поддержка всех основных графических примитивов
- Управление ориентацией и яркостью
- Оптимизированная работа с SPI

### 2. Драйвер тачскрина XPT2046
- Калибровка тачскрина
- Фильтрация дребезга и усреднение измерений
- Поддержка измерения давления (Z-ось)
- Автоматическое определение касания

### 3. Генератор сигнала джаммера
- **4 режима работы:**
  - Continuous (непрерывное излучение)
  - Sweep (сканирование по диапазону частот)
  - Burst (импульсный режим)
  - Noise (широкополосный шум)
- **5 режимов частоты:**
  - WiFi 2.4GHz (2.4 - 2.4835 GHz)
  - WiFi 5GHz (5.15 - 5.825 GHz)
  - Bluetooth (2.402 - 2.480 GHz)
  - Cellular (700 MHz - 2.6 GHz)
  - Custom (пользовательская частота)
- Управление мощностью (0-100%)
- LED индикация состояния

### 4. Система настроек
- Сохранение в JSON файл
- Автоматическое применение настроек при запуске
- Автосохранение при изменении параметров
- Калибровка тачскрина

### 5. Пользовательский интерфейс
- Тач-интерфейс с кнопками
- Две страницы: главная и настройки
- Статус бар с информацией
- Цветовая индикация состояния

## Требования к оборудованию

### Аппаратное обеспечение:
- Raspberry Pi Pico (или совместимая плата)
- Дисплей ILI9341 2.4" с тачскрином XPT2046
- Внешний генератор сигнала (подключён к GP28)
- LED индикатор (подключён к GP25)

### Подключение:
```
Дисплей ILI9341:
  SCK  -> GP18 (Pin 24)
  MOSI -> GP19 (Pin 25)
  MISO -> GP16 (Pin 21)
  CS   -> GP17 (Pin 22)
  DC   -> GP20 (Pin 26)
  RST  -> GP21 (Pin 27)
  BLK  -> GP14 (Pin 19)

Тачскрин XPT2046:
  CS   -> GP22 (Pin 29)
  IRQ  -> GP15 (Pin 20)

Джаммер:
  Signal -> GP28 (Pin 34)
  LED    -> GP25 (встроенный LED)

Аппаратные кнопки (опционально):
  Power      -> GP2 (Pin 4)
  Mode       -> GP3 (Pin 5)
  Power Up   -> GP4 (Pin 6)
  Power Down -> GP5 (Pin 7)
```

## Использование

### Главный экран:
- **START/STOP** - включение/выключение джаммера
- **MODE** - переключение режима работы
- **+/-** - регулировка мощности
- **SETUP** - переход к настройкам

### Экран настроек:
- Просмотр текущих параметров
- **SAVE** - сохранение настроек
- **BACK** - возврат на главный экран

## Режимы работы джаммера

### 1. Continuous (Непрерывный)
- Постоянное излучение на выбранной частоте
- Максимальная эффективность для точечного подавления

### 2. Sweep (Сканирование)
- Автоматическое сканирование по диапазону частот
- Подавление нескольких каналов одновременно
- Настраиваемая скорость сканирования

### 3. Burst (Импульсный)
- Импульсное излучение с паузами
- Экономия энергии
- Сложность для обнаружения

### 4. Noise (Шум)
- Широкополосный псевдослучайный сигнал
- Максимальное покрытие диапазона
- Высокая эффективность против цифровых систем

## Конфигурация

Все параметры настраиваются в файле `config.py`:

```python
# Частоты (Гц)
WIFI_24GHZ_MIN_FREQ = 2_400_000_000
WIFI_24GHZ_MAX_FREQ = 2_483_500_000

# Мощность (%)
JAMMER_MAX_POWER_LEVEL = 100
JAMMER_DEFAULT_POWER_LEVEL = 50

# Параметры режимов
JAMMER_SWEEP_STEP_HZ = 1_000_000      # Шаг сканирования
JAMMER_SWEEP_DELAY_MS = 10            # Задержка между шагами
JAMMER_BURST_ON_MS = 100              # Длительность импульса
JAMMER_BURST_OFF_MS = 50              # Пауза между импульсами
```

## Совместимость с C-версией

Данная MicroPython версия сохраняет полную функциональность оригинальной C-версии:

- **Тот же интерфейс управления** (тачскрин + кнопки)
- **Те же режимы работы** (continuous, sweep, burst, noise)
- **Те же диапазоны частот** (WiFi 2.4/5GHz, Bluetooth, Cellular)
- **Та же система настроек** с автоматическим сохранением
- **Совместимая конфигурация** пинов и параметров

## Ограничения MicroPython

1. **Производительность**: MicroPython медленнее C, но достаточно для данного приложения
2. **Память**: Ограниченная память требует оптимизации
3. **Точность таймеров**: Менее точные таймеры по сравнению с C

## Лицензия

Проект распространяется под той же лицензией, что и оригинальная C-версия.
